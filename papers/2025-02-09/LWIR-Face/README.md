# LWIR-Face: Long-Wave Infrared Face Recognition in Outdoor Environments

## 论文信息
- **arXiv ID**: 2502.08845
- **发布日期**: 2025-02-09
- **分类**: 人脸识别
- **来源**: arXiv

## 作者
Zhi Chen, Guang Wang, Jin Li

## 论文简介
LWIR-Face 研究了长波红外（LWIR）人脸识别在户外环境中的应用。针对长波红外图像的特点，该方法提出了多尺度特征融合和注意力机制，在户外人脸识别任务中显著提升了识别准确率。

## 关键词
LWIR, infrared, outdoor face recognition
