# Feature2Face: Recovering Face Images from Deep Embedding Space

## 论文信息
- **arXiv ID**: 2502.11567
- **发布日期**: 2025-02-13
- **分类**: 人脸生成
- **来源**: arXiv

## 作者
Minghua Liu, Jian Sun, Xiaojie Wang

## 论文简介
Feature2Face 提出了一种从人脸深度特征嵌入恢复高质量人脸图像的方法。该方法使用生成对抗网络将压缩的人脸特征解码为逼真的人脸图像，可用于隐私保护的人脸识别系统和人脸特征可视化。
