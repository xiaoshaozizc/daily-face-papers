# LightFace: Efficient Lightweight Face Detection Network for Mobile Devices

## 论文信息

- **arXiv ID**: 2501.06234
- **发布日期**: 2025-01-11
- **分类**: 人脸识别
- **关键词**: face detection, mobile, lightweight model

## 作者

Cheng Cui, Kai Wang, Yuwen Wu, Jianshu Li

## 论文简介

针对移动端实时人脸检测的需求，LightFace设计了一种高效的轻量化检测网络。该方法采用深度可分离卷积和特征融合策略，在保证检测精度的同时大幅降低模型参数量和计算量。实验表明，LightFace在移动设备上可达到50fps以上的检测速度，同时保持高准确率。
