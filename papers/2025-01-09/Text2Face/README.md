# Text2Face: Text-Guided Face Generation Using Hierarchical Diffusion Model

## 论文信息

- **arXiv ID**: 2501.04892
- **发布日期**: 2025-01-09
- **分类**: 人脸生成
- **关键词**: text-to-face, diffusion model, text-guided

## 作者

Wenqi Wang, Yuxuan Wang, Jiankang Deng, Xiaokang Yang

## 论文简介

Text2Face提出了一种基于分层扩散模型的文本引导人脸生成方法。该系统能够根据自然语言描述生成相应的人脸图像，并保持面部细节的一致性。研究还构建了一个大规模文本-人脸配对数据集用于训练，在定量和定性实验中均取得了优异表现。
