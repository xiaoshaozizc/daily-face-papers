# Face-Voice Biometric Fusion for Person Authentication

## 论文信息
- **arXiv ID**: 2502.15023
- **发布日期**: 2025-02-20
- **分类**: 人脸识别
- **来源**: arXiv

## 作者
Wei Liu, Jian Yang, Xiao Wang

## 论文简介
本文提出了一种将人脸识别与语音识别融合的多模态身份认证方法。通过深度神经网络学习人脸和语音特征的联合表示，在LFW和VoxCeleb数据集上取得了优异的识别准确率。

## 关键词
