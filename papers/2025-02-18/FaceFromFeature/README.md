# FaceFromFeature: Photorealistic Face Reconstruction from Embedding Features

## 论文信息
- **arXiv ID**: 2502.13876
- **发布日期**: 2025-02-18
- **分类**: 人脸生成
- **来源**: arXiv

## 作者
Jianhua Lin, Cheng Wang, Xiaoming Liu

## 论文简介
FaceFromFeature 提出了一种从人脸嵌入特征重建高保真人脸图像的方法。该方法利用预训练的人脸编码器提取特征，再通过扩散模型从特征重建 photorealistic 人脸图像。在CelebA-HQ和FFHQ数据集上验证了方法的有效性。

## 关键词
