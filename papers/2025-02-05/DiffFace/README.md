# DiffFace: High-Fidelity Face Synthesis with Diffusion Probabilistic Models

## 论文信息

- **arXiv ID**: 2502.01965
- **发布日期**: 2025-02-03
- **分类**: 人脸生成
- **关键词**: face generation, diffusion model, face synthesis

## 作者

Yue Qian, Jie Zhang, Weiming Jiang, Jue Wang

## 论文简介

DiffFace是一种基于扩散概率模型的高保真人脸合成方法。该研究提出了新的条件去噪网络和Identity-Preserving损失函数，能够在保持人物身份特征的同时生成高质量的人脸图像。实验结果显示，DiffFace在FFHQ和CelebA-HQ数据集上取得了优异的FID分数。
