# LiveFace: Liveness Detection for Face Recognition Against Presentation Attacks

## 论文信息

- **arXiv ID**: 2502.03287
- **发布日期**: 2025-02-05
- **分类**: 人脸识别
- **关键词**: face recognition, liveness detection, face anti-spoofing

## 作者

Jian Wang, Shangman Zhou, Zheng Xia, Zicheng Liu

## 论文简介

本文提出了一种名为LiveFace的人脸活体检测框架，用于检测人脸识别系统面临的呈现攻击（presentation attacks）。该方法结合了多尺度特征提取和时序建模，能够有效区分真实人脸和屏幕照片、打印照片、3D面具等攻击手段。在多个基准数据集上的实验表明，该方法达到了最先进的检测精度。
