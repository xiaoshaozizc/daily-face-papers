# UltraFace: Ultra-Lightweight Face Detection for Real-Time Applications

## 论文信息
- **arXiv ID**: 2502.13567
- **发布日期**: 2025-02-17
- **分类**: 人脸识别
- **来源**: arXiv

## 作者
Hao Li, Feng Liu, Wei Zhang

## 论文简介
UltraFace 设计了一种超轻量级的人脸检测网络，专门针对移动端和嵌入式设备的实时检测需求。通过知识蒸馏和模型剪枝技术，在保持高检测精度的同时将模型大小压缩至1MB以下。

## 关键词
face detection, lightweight, real-time
