# ThermalFace: Infrared Face Recognition with Deep Thermal Imaging

## 论文信息
- **arXiv ID**: 2502.13322
- **发布日期**: 2025-02-17
- **分类**: 人脸识别
- **来源**: arXiv

## 作者
Wei Chen, Yong Wang, Jie Liu

## 论文简介
ThermalFace 研究了基于深度学习的红外人脸识别方法。红外图像不受光照影响，适合在黑暗环境中进行人脸识别。该方法提出了新的热红外人脸特征提取网络，在OTIR数据集上取得了先进的识别性能。

## 关键词
infrared face recognition, thermal imaging, deep learning
