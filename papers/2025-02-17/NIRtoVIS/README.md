# NIRtoVIS: Cross-Modal Face Recognition from Near-Infrared to Visible Light

## 论文信息
- **arXiv ID**: 2502.12541
- **发布日期**: 2025-02-15
- **分类**: 人脸识别
- **来源**: arXiv

## 作者
Qiang Sun, Fang Liu, Hao Zhao

## 论文简介
NIRtoVIS 提出了一种近红外到可见光的人脸跨模态识别方法。该方法通过GAN生成伪可见光图像，缩小近红外和可见光人脸图像之间的模态差异，在CASIA NIR-VIS 2.0数据集上取得了优异的跨模态识别效果。

## 关键词
NIR to VIS, cross-modal, face recognition
