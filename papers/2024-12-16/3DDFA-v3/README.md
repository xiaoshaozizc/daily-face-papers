# 3DDFA-v3: Robust 3D Face Alignment and Reconstruction in the Wild

## 论文信息

- **arXiv ID**: 2412.12456
- **发布日期**: 2024-12-16
- **分类**: 人脸生成
- **关键词**: 3D face, face alignment, face reconstruction

## 作者

Jianzhu Guo, Xiangyu Zhu, Jie Lei, Stan Z. Li

## 论文简介

3DDFA-v3提出了在自然场景下鲁棒的3D人脸对齐与重建方法。该方法采用混合网络架构，结合CNN的特征提取能力和Transformer的全局建模能力，能够准确估计68个关键点和3D面部形状。在多个野外数据集上的实验表明，该方法在遮挡、极端光照等困难条件下具有很强的鲁棒性。
