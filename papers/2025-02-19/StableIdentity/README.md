# StableIdentity: Preserving Identity in Face Editing with Stable Diffusion

## 论文信息
- **arXiv ID**: 2502.14201
- **发布日期**: 2025-02-19
- **分类**: 人脸生成
- **来源**: arXiv

## 作者
Ming Zhang, Yuan Chen, Lei Wang

## 论文简介
StableIdentity 提出了一种基于稳定扩散模型的人脸编辑方法，能够在修改人脸属性（如年龄、表情、发型）的同时保持人物身份特征不变。该方法引入了身份保持损失和注意力机制，有效解决了属性编辑中的身份泄露问题。

## 关键词
face editing, stable diffusion, identity preservation
